# Event Organizer Bot

## Мета проєкту
Консольний чат-бот на Python для організації подій.

## Логіка роботи
Бот дозволяє:
- додавати події
- переглядати список подій
- фільтрувати події
- знаходити найближчу подію

Події зберігаються у файлі JSON.

## Структура даних
Кожна подія має:
- name
- date
- time
- category

## Запуск програми

python bot_zemlyacov.py
